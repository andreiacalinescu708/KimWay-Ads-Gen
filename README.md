# Kimiway Ads Generator - Versiunea Simplă

## Setup

1. Copiază `.env.local.example` în `.env.local` și adaugă API key-ul KIMI
2. `npm install`
3. `npm run dev`
4. Deschide http://localhost:3001

## Deploy pe Railway

1. Push pe GitHub
2. În Railway: New Project → Deploy from GitHub
3. Adaugă variabila `KIMI_API_KEY`
4. Deploy!

## Funcționalități

- ✅ Formular pentru produs
- ✅ Generează script cu KIMI AI
- ⏳ Generează video cu Runway (următorul pas)
